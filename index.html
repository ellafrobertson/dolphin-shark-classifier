Dolphin vs Shark Classifier

An AI-Powered Marine Life Classification Model

About the Project

This project uses AlexNet, a convolutional neural network, to classify images of dolphins and sharks with high accuracy. The model was trained on a custom dataset and achieved 85% validation accuracy.

Features

  • Custom-trained AlexNet model
  • Dataset of dolphin and shark images
  • Real-world applications in marine conservation

How It Works

Upload an image of a dolphin or shark to the model to get a prediction. The model uses CNNs to extract features and classify the image with high precision.
